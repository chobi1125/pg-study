## 使い方

インストール

npm i -D

起動・自動トランスパイル

gulp

作成ディレクトリ

src

出力先ディレクトリ

dist

## firebase

firebase側でプロジェクト生成

firebase init で紐づけ

起動

firebase serve

デプロイ

firebase deploy
